# Final Year BSc Project
In this project, we use statistical techniques and neural networks to create forecasts. This includes multiple linear regressions (MLR), multilayer perceptrons (MLP), convolutional neural networks (CNN), and long short-term memory (LSTM).

The datasets used in this project are available upon request from the project supervisor.

All the code used for this project is stored in the file named "script-for-final-year-project.py".
